Problem Statement

This is a 12 hr programming paradise for insomniacs where they will be served with mind boggling coding and algorithmic challenges to solve. There will be flexibility to use any programming language of their choice.

Rules and Regulations

  • Partcipate should register at www.axisvnit.org
  • The Contest will be conducted on HackerEarth. You can register for the event there.
  • The Contest rules will be displayed on the contest page.
  • Participants will need Axis id and HackerEarth id to participate.
  • The contest duration will be 12hrs.
  • The contest begins on 21 Feb 5:00 pm and ends on 22 Feb 5:00 am.
  • In case of any discrepancy, the decision of the event managers will be final and binding.

    • SCORING

  • Scoring is based on Speed, Accuracy and Efficiency of the code.
  • Each question has predetermined score.
  • The participants would get partial score only after passing certain number of test cases. To get the full score, all test cases have to be passed efficiently.
  • In case of same scores of participants, the tie is broken by sum of time participant took in submitting the last solution resulting the highest score.
